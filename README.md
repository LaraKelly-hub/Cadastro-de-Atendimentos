# Cadastro-de-Atendimentos



📞 Sistema de Registro de Atendimentos em Python

Este projeto foi desenvolvido em Python com o objetivo de simular um sistema simples de registro e análise de atendimentos ao cliente. O programa permite cadastrar atendimentos, validar as informações inseridas pelo usuário e gerar estatísticas automaticamente.

🚀 Funcionalidades
Cadastro de múltiplos atendimentos.
Validação de entradas utilizando try/except.
Verificação de tipos de atendimento permitidos:
Suporte
Reclamação
Dúvida
Registro das informações utilizando listas e dicionários.
Cálculo do tempo médio de atendimento.
Identificação dos clientes com atendimento acima da média.
Contagem automática da quantidade de atendimentos por categoria.
Exibição do atendimento com maior tempo de duração.
🛠️ Tecnologias utilizadas
Python 3
📚 Conceitos praticados

Durante o desenvolvimento deste projeto foram aplicados diversos conceitos fundamentais da linguagem Python, como:

Estruturas de repetição (for e while)
Estruturas condicionais (if, elif e else)
Tratamento de exceções (try/except)
Listas
Dicionários
List Comprehension
Validação de dados
Manipulação de coleções
Cálculos estatísticos simples
🎯 Objetivo

O principal objetivo deste projeto foi praticar lógica de programação, validação de dados e manipulação de estruturas de dados em Python, simulando um cenário real de controle de atendimentos em uma central de suporte.
